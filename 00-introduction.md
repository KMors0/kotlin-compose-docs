# Введение: Схождение трёх технологий

> Мобильная и кроссплатформенная разработка переживает трансформацию, определяемую не просто новыми фреймворками, а фундаментальным сдвигом в подходе к построению пользовательских интерфейсов. Конвергенция языка Kotlin, декларативного фреймворка Compose Multiplatform и дизайн-системы Material Design 3 представляет собой уникальный технологический стек, способный консолидировать разработку для Android, iOS, десктопа и веба в единой кодовой базе без компромиссов в производительности и качестве пользовательского опыта.

---

## Почему именно этот стек?

По данным отчёта JetBrains Developer Ecosystem Survey за 2025 год, доля разработчиков, использующих Kotlin Multiplatform (KMP), выросла с 7% до 18% за два года, а по данным KMPShip — с 12% до 23% за 18 месяцев в enterprise-сегменте. Крупнейшие компании — Netflix, McDonald's, Cash App, Philips, Google Workspace — уже внедрили KMP в продакшен для обработки реальных платежей, медицинских данных и корпоративных инструментов. Релиз Compose Multiplatform 1.8.0 в мае 2025 года закрепил стабильную поддержку iOS, а версия 1.11.0 (май 2026) довела iOS и Web до зрелости, добавив unified `@Preview`, Navigation 3 на non-Android targets и refreshed UI testing.

**Тезис данного руководства:** Стек Kotlin + Compose Multiplatform + Material Design 3 в 2026 году является наиболее перспективным решением для организаций, стремящихся максимизировать повторное использование кода, сохранить нативную производительность и обеспечить визуальную согласованность пользовательского интерфейса на всех целевых платформах.

## Краткий обзор технологий

### Kotlin

Kotlin — статически типизированный мультипарадигменный язык программирования, разработанный компанией JetBrains. Текущая стабильная версия — Kotlin 2.4.0 (май 2026). Язык работает поверх JVM и поддерживает компиляцию в машинный код через Kotlin/Native, в JavaScript/WebAssembly через Kotlin/JS и Kotlin/Wasm. Ключевые рубежи: релиз Kotlin 2.0 (май 2024) с новым компилятором K2 (до 94% ускорение чистых сборок), Kotlin 2.4 с поддержкой Swift packages как зависимостей и Swift Export (Alpha), Kotlin 2.4.20-Beta1 с coroutine stack trace recovery в standard library.

### Compose Multiplatform

Фреймворк от JetBrains для декларативного кроссплатформенного UI. Ключевые вехи:
- **1.8.0 (май 2025)** — стабильная поддержка iOS.
- **1.9.0 (сентябрь 2025)** — бета-поддержка Web (Kotlin/Wasm).
- **1.10.0 (апрель 2026)** — unified `@Preview` для всех платформ, Navigation 3 на non-Android targets.
- **1.11.0 (май 2026)** — refreshed UI testing, улучшения iOS/Web, Skia M144.

Полный паритет API с Jetpack Compose для Android.

### Material Design 3

Обновлённая дизайн-система Google с динамическим цветом (Dynamic Color), расширенной типографикой и адаптивными компонентами. M3 Expressive (стабилизирован в `compose-material3` 1.4.x в 2026) — расширение с вибрантными цветовыми схемами, MotionScheme (физически корректные motion-токены) и новыми компонентами: LoadingIndicator, SplitButton, FloatingActionButtonMenu.
