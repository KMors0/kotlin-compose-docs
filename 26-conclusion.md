# Заключение и перспективы

> Стек Kotlin + Compose Multiplatform + Material Design 3 представляет собой в 2025–2026 годах зрелый, промышленно готовый технологический стек для кроссплатформенной разработки.

---

## Ключевые выводы

### 1. Kotlin — зрелый фундамент

Kotlin — продуктивный язык с передовой системой типов, встроенными корутинами и новым компилятором K2, обеспечивающим до 94% ускорение компиляции. Является идеальным фундаментом для кроссплатформенной разработки благодаря KMP. Философия прагматизма, безопасности и лаконичности делает Kotlin одним из самых продуктивных языков для промышленной разработки.

### 2. Compose Multiplatform — уникальная кроссплатформенность

Единственный фреймворк, позволяющий использовать один и тот же декларативный UI-код на Android, iOS, Desktop и Web с полной паритетностью API. Статус iOS stable (май 2025) и Web beta (сентябрь 2025) расширяют адресное пространство применимости. Рендеринг через Skia обеспечивает нативную производительность и поддержку 120 Гц дисплеев.

### 3. Material Design 3 — визуальная согласованность

Обеспечивает визуальную согласованность, доступность и адаптивность на всех платформах. M3 Expressive добавляет выразительность через расширенные формы, motion-физику и вибрантные цветовые схемы. Динамический цвет на Android 12+ создаёт уникальный пользовательский опыт.

### 4. Инкрементальное внедрение — ключевое преимущество

Организации могут начать с вынесения общей бизнес-логики и постепенно переносить UI, минимизируя риски. Это фундаментальное отличие от Flutter и React Native, которые требуют полного перехода на свою парадигму.

---

## Перспективы развития

Ожидается дальнейшее улучшение по нескольким направлениям:

- **Производительность iOS:** оптимизация Metal-бэкенда, цель — «абсолютно плавная производительность» на всех устройствах
- **Стабилизация Web:** выход из beta-статуса, улучшение initial load time, расширение интеграции с браузерными API
- **M3 Expressive:** дальнейшее развитие дизайн-системы с новыми компонентами и возможностями кастомизации
- **Экосистема:** рост количества сторонних библиотек с KMP-поддержкой, упрощение интеграции с платформенными SDK
- **Инструменты:** улучшение поддержки KMP в Android Studio и IntelliJ IDEA, новые возможности отладки и профилирования

С учётом текущей траектории развития, Compose Multiplatform имеет потенциал стать доминирующим кроссплатформенным фреймворком для команд, уже работающих с Kotlin и Android.

---

## Рекомендация

Для команд с сильной Kotlin/Android-экспертизой, стремящихся к кроссплатформенности без потери нативного качества, стек Kotlin + Compose Multiplatform + Material 3 является оптимальным выбором в 2025–2026 годах. Для проектов с критичными веб-требованиями (SEO, контентные платформы) рекомендуется рассмотреть гибридный подход: Compose Multiplatform для мобильных и десктопа + специализированный веб-фреймворк для веба.
